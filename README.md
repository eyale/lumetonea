# Lumetonea
Skin tone & undertone analyzer with personalized palettes (SwiftUI, on-device).

## Tech
SwiftUI · Vision · Core Image · Accelerate (k-means)

## Run
- Xcode 15+
- iOS 16+ target
- Open `Lumetonea.xcodeproj` (or `.xcworkspace` if using SPM/CocoaPods)

## Roadmap (MVP)
- Photo import → Face landmarks → Skin sampling → Undertone → Palette → Export PDF
